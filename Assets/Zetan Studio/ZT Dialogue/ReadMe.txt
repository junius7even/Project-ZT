ZT Dialogue

ZT Dialogue is a node-based dialogue plugin, design complex dialogues with simple connections, such as designing branches, etc., which is very convenient for constructing plot dialogues.
In this plugin you can:
â— Use a graphical editor to edit dialogues, and you can  undo and redo the actions you have made;
â— Use sentence nodes to set up the speech contents, and the typing animation speed, you can choose to display the portrait of the interlocutor in the dialogue, and play the voice corresponding to each sentence;
â— Use decorator nodes to implement functions such as bolding the option title text or color them with random colors;
â— Use condition nodes to implement functions such as hiding some options or switching different next sentence according to conditions, etc.;
â— Use block nodes to implement functions such as clicking an option and switching to the next sentence only when conditions are met;
â— use recursive nodes to return to the specified sentence in the dialogue;
â— Use external-options nodes to mess up the order of options, easily construct Q&A dialogues;
â— Use other-dialogue nodes to insert a sub-dialogue into the current dialogue, and return to the original dialogue when the sub-dialogue ends;
â— Use dialogue events to trigger specific events before or after some sentences are spoken, which can be used to implement functions such as mirroring and playing character actions;
â— Use the built-in save system to save and load the dialogue progress, and even game progress if you want;
â— Use the built-in UI window framework to build the game UI;
â— Use the list-based interaction system to interact, just like "Genshin Impact";
â— Insert the strings called "keyword ID" into the dialogue text, and they will be converted into their corresponding names;
â— Implement the multi-language switching function of the game text, and directly edit the translations in the form of a table in Unity, or import the translations from Excel file;
â— Easily extend the node type to implement the desired node function;
â— Easily extend the built-in dialogue window, such as use its option list as a quest list to show quests.
All codes of this plugin are in the form of ".cs" files instead of ".dll", so you can modify everything as you want, and they are also very simple, you can even make this plugin yourself if you spend some time, I just make it in advance for youğŸ˜„.
